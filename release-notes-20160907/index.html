2016-09-07 扫码支付上线说明

发表于 2016-09-07   |   分类于 更新说明

好农场扫码支付隆重上线。更好的客户体验，同时给农场主带来更强的便利性。支付码可以用于好农场市集，也可以用于农场日常的收款。推荐各位农场主使用。

二维码获取

农场主登录好农场后台后，可以在这里找到：［农农场］／［农场资料］／［基本资料］，如果需要打印，可以点击二维码打开大分辨率的版本。

支付流程

  1. 在微信会话页面中，点击［扫一扫］
  2. 扫描二维码
  3. 输入金额
  4. 选择微信支付
  5. 输入支付密码
  6. 完成！

支付确认

  1. 农场主手机短信通知

    支付成功后，农场主会收到一条到账通知，内容类似如下：

    【好农场】有肉农场有一笔新的收入，金额：300.0元，来自-线下客户-扫码

  2. 后台查询与提现

    在［农场］／［收入/提现］栏目下，可以查看到最新的扫码支付记录。

    扫码支付入账的金额同套餐购买／充值的金额一样，可以发起提现操作。
