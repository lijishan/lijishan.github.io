【教程】首次会员配送流程

发表于 2016-01-21   |   分类于 新手指引

如果你是第一次使用好农场来进行会员配送，那么你只需要按照下面的流程一步步操作，就可以完成一次会员配送了。

操作流程

  1. 新增会员
  2. 设置会员菜单
  3. 会员点菜
  4. 开始配送

流程介绍

  1. 新增会员
    新增会员之前，需要先 新增会员配送日 和 会员套餐，完成以上操作后，可以进行 新增会员 操作

  2. 设置会员菜单

    设置会员菜单之前，需要先在商品库中 新增商品（蔬菜或附加商品），新增商品后，可以 设置会员可选菜单和附加商品

  3. 会员点菜

    不管农场是否允许会员点菜，农场需要先找到对应的配送日，并“开始配送计划”。配送计划生成后，农场就可以 开放点菜 了，农场开放点菜后，会员的app就会收到点菜提醒，并开始点菜。到截止时间后，农场可以 结束点菜。

  4. 开始配送

    结束点菜后，就可以打印送货单或销售单并 开始配送 了。配送完成 后，系统会将该配送日的订单批量处理为“配送完成”，整个配送过程就结束了，农场可以到“全部会员订单”中查询历史订单。
