开始点菜

发表于 2016-01-21   |   分类于 会员订单

执行完 开始配送计划，就可以开始点菜了。

一、操作路径

好农场管理后台-订单-会员订单

二、操作流程

  1. 开始点菜
  2. 查看点菜状态

三、流程介绍

  1. 开始点菜

    点“开始点菜”后，当前配送日所有的会员订单状态会变为”用户点菜“，这时会员也可以在“好农场app”上点菜了。

    关于好农场app的使用介绍请查阅：会员如何使用好农场app?

  2. 查看点菜状态

    开始点菜后，如果会员还未点菜，则会员订单的点菜状态为未点菜。

    提示：

    1. 如果会员通过好农场app点菜，则会员订单的点菜状态为app点菜；
    2. 农场可以通过 修改订单 在后台给会员点菜，在修改订单页勾选“已点菜”后，会员订单的点菜状态为后台点菜；
    3. 点击”展开高级搜索“可以通过点菜状态来筛选订单。

开始配送计划

发表于 2016-01-21   |   分类于 会员订单

一、操作路径

好农场管理后台-订单-会员订单

二、操作流程

  1. 选择配送日
  2. 确认可选蔬菜
  3. 确认附加商品

三、流程介绍

  1. 选择配送日

    点击要开始配送的配送日，例如“星期二”，具体的配送日期系统会自动更新。点“开始配送计划”就可以开始本次配送了。

  2. 确认可选蔬菜

    确认一下之前在“会员商品“中设置的菜单，在这里可以直接修改库存和限购数量，也可以点右上角的“调整蔬菜”去可选蔬菜页面调整。

  3. 确认附加商品

    执行“下一步”后会确认一下原先设置的附加商品，同样也可以调整。

    会员套餐订购数量：指会员在订购套餐时设置的“套餐附加商品”，如果有会员在套餐中订购了某一商品，则在“确认生成”后，系统会自动给该会员分配这个商品（除非库存不足）。

    确认生成后，系统会给所有该配送日期没设置停配的会员生成一个配送订单。

    提示:

    1. 在”开始点菜“之前，点击”取消本次配送“可以取消已生成的配送计划。
    2. 点击“未配送会员”，可以查看本次配送有哪些会员设置了停配，并在弹出窗中为会员恢复本次配送和创建订单。

设置会员商品

发表于 2016-01-21   |   分类于 商品管理

一、操作路径

好农场管理后台-商品-会员商品

二、操作流程

  1. 上架可选蔬菜
  2. 上架附加商品

三、流程介绍

  1. 上架可选蔬菜

    在会员商品下选择”可选蔬菜“，可以通过”上架“或”复制上次菜单“的方式上架可选蔬菜。

    复制上一次菜单：点击后可以将上一次配送时上架的蔬菜直接复制到当前配送日。

    提示：

    1. 在上架蔬菜前应先切换到要执行配送的配送日，如下图选中的是“星期二”。
    2. 限购数量为0时表示不限购。

    在上架蔬菜的弹出框中，只能选择新增商品时商品类目为蔬菜 的商品。

  2. 上架附加商品

    附加商品和可选蔬菜的区别在于：

    附加商品里的商品和库存对于每个配送日来说都是可选的，而可选蔬菜则是一个配送日对应一份菜单。

新增商品

发表于 2016-01-21   |   分类于 商品管理

一、操作路径

好农场管理后台-商品-商品库

二、操作流程

  1. 设置商品类目
  2. 设置商品基本信息
  3. 添加商品详情

三、流程介绍

  1. 设置商品类目

    在商品库中点”新增商品“，进入商品类目选择页，请根据要添加的商品选择适合的分类。

  2. 设置商品基本信息

    • 在设置商品基本信息时，可以对商品进行分组，便于按分组管理商品，查阅：商品分组功能
    • 在新增了会员套餐后，可以对商品设置套餐价，每一个套餐可以对应一种价格。
    • 可以给商品设置运费模板，运费模板介绍请查阅：运费模板功能 ， 运费模板暂时只对零售商品生效。

  3. 编辑商品详情

    可以通过编辑框编辑商品的图文介绍信息。

如何申请提现

发表于 2016-01-21   |   分类于 农场设置

一、操作路径

好农场管理后台-农场-收入/提现

二、操作流程

  1. 在后台设置提现银行卡
  2. 提交提现申请，等待提现处理

三、流程说明

  1. 设置提现银行卡

    点击账户余额旁边的 "提现“ 按钮，如果未设置过银行卡号，则进入设置提现账号。

  2. 输入提现金额，点击”提交”就可以了，目前提现是T+2天处理，5个工作日内到账，节假日顺延。

  3. 当提现状态为“提现成功”时，即表示提现金额已经到账。

新增会员

发表于 2016-01-21   |   分类于 会员管理

新增会员有两种途径：一种是新增单个会员，一种是批量导入会员。

一、操作路径

好农场管理后台-会员-配送会员

二、新增单个会员

  1. 在左上角点“新增会员”按钮

  2. 添加会员基本信息

    可透支额度（元）：指套餐余额消费完后还可以透支消费的金额

  3. 设置套餐信息

    单次配送要求：随套餐的要求而变化

    配送日：一个会员至少对应一个配送日和一条配送线路，可设置多个配送日。

    套餐附加商品：指每次点菜前，如果该附加商品有库存，系统会自动把该商品分配到会员的购物车里。

三、批量导入会员

在“配送会员”页左上角点“导入”，按导入指引和模板提醒导入即可
