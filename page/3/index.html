好农场零售功能使用须知

发表于 2016-01-21   |   分类于 好农场简介
Q：好农场零售功能的定位是什么样的？目标消费群体是哪些？

A：好农场零售功能的定位是一个以本地农场为主的生态农产品电商平台，软件根据消费者所在位置自动向消费者推荐周边的农场和农产品。目标消费群体是城市里对食品安全有需求的人群（如孕妇、有老人或小孩的家庭、年轻白领等）

Q：好农场零售平台农场入驻的标准或要求是什么？

A：主要有以下几点要求：

  1. 愿意公开并承诺农场的生产方式，例如种养植标准、加工标准；
  2. 接受好农场工作人员或消费者随时到农场参观考察，对生产进行监督；
  3. 可以独立发货、并能够保证向消费者提供有品质保障的农产品；
Q：好农场电商平台和一般的电商有什么区别？流量从哪里来？

A：好农场电商平台是一个只做生态农产品的平台，旨在帮助用心做农业的农人获得合理的回报。平台上的每一个农产品都会有对应的生产者及生产承诺，好农场会根据农场做出的承诺随机抽检，消费者也可以对违反承诺的商品进行举报。好农场平台的流量来自于线下的口碑传播及我们在农夫市集及其它场所做的宣传和推广活动。

会员宅配管理系统使用须知

发表于 2016-01-21   |   分类于 好农场简介
Q：好农场会员宅配管理系统适应于哪种类型的农场？

A：可以从以下几个条件来判断你的农场是否适合使用：

  1. 按预付款分不同的会员套餐（或储值卡）类型，不同的套餐可以对应不同的商品价格；
  2. 会员按固定的配送日每周配送，要求会员每次配送不少于n斤蔬菜或n份蔬菜；
  3. 每次配送按会员所选的蔬菜或商品金额从预付款中扣款结算，如果未选菜，则由农场安排配送。

简单来说就是：

  1. 会员是否按预付款购买储值套餐
  2. 会员是否按固定配送日每周配送
  3. 会员结算是否从储值卡中扣款结算

如果你的农场业务符合上述条件，欢迎联系我们来帮助你使用。

Q：我的农场是按预付款购买配送次数的套餐模式，每次配送按次结算（例如：预付3000购买蔬菜配送30次，每周配送一次或两次，每次配送n斤），可以用你们的系统吗？

A：我们正在做这方面的开发，如果农场急需一套会员管理系统，则可以将按次数结算的方式转换为按金额结算（例如3000元配送送30次，则每次配送按100元结算），且这种方式有助于农场在销售蔬菜的同时销售其它附加商品。

Q：好农场会员宅配管理系统和微店、有赞的会员管理有什么不同？

A：好农场是针对会员宅配制农场开发的会员管理系统，主要特点有：

  1. 农场可以按配送日、配送线路来划分会员，并给不同配送日的会员设置不同的可选菜单及点菜时间；
  2. 可以对会员进行停配送管理、套餐账户充值、消费记录查询、套餐退订等操作；
  3. 可以根据配送日只处理本周的配送订单，打印备货单、送货单及会员销售单等；
  4. 可以根据时间段统计会员的配送情况、消费情况以及农场商品的销售情况。
Q：好农场会员宅配管理系统和电商平台（零售功能）会冲突吗？是否会造成农场已有会员的流失？

A：农场使用好农场来管理会员后，会员打开好农场app将进入农场专属的会员页，电商平台的入口被隐藏在app首页顶部的农场名下面，会员需要点击才能切换进去。在已上线的农场中我们得出的统计数字是只有1%的会员会点击这里进入电商平台，对会员流失产生的影响可以说是微乎其微。

A：好农场是一套为会员宅配制农场（包括CSA农场）打造的会员宅配管理系统，同时也是一个生态农产品垂直电商平台。农场可以根据自己的业务范围选择使用好农场来管理会员或在平台上销售农产品；好农场提供网页版的农场管理后台供农场管理会员或零售订单，提供好农场app供会员点菜、充值或消费者购买商品。

好农场支付方式及结算规则

发表于 2016-01-21   |   分类于 好农场简介

一、支付方式

好农场目前支持以下两种在线支付方式：

  1. 微信支付

  2. 支付宝支付

    农场会员充值或消费者购买商品时，可以选择以上方式向农场支付，支付金额统一进入好农场结算账户，农场通过管理后台发起提现申请与好农场结算款项。好农场将给农场补贴支付时产生的手续费用0.6%以及提现时的手续费用。

    说明：目前微信支付和支付宝支付的商户结算费率均为 0.6%
    点击查看： 微信支付结算费率 ， 支付宝支付结算费率

（注：农场会员宅配系统中的会员订单均只能使用会员的储值余额支付，余额不足时需进行充值，与上述的支付方式为不同类型）

二、会员充值或买家下单后，资金如何结算？如何提现？

  1. 会员充值成功后，在“农场”——“收入/提现”里会产生一条充值记录，充值的金额将实时进入农场的账户余额中，农场可以随时申请提现。

  2. 买家购买零售商品并支付后，以下两种情况满足任意一种结算金额可以入账提现：

    • 农场在订单发货T+7天后交易金额将结算入账可提现（从发货当天开始计算，7天结算精确到天）

      例：交易订单在7月17号发货，则7月24号交易金额结算入账。

    • 农场在订单发货后，买家在好农场app上进行了“确认收货”操作，交易金额将结算入账可提现。

      例：交易订单在7月17号发货，7月20号买家确认收货，则交易金额结算入帐。

  3. 提现前农场需添加结算账户，具体查看：如何进行提现操作？ ；提现周期为T+2个工作日，具体到账时间以银行的处理时间为准。
