新增会员

发表于 2016-01-21   |   分类于 会员管理

新增会员有两种途径：一种是新增单个会员，一种是批量导入会员。

一、操作路径

好农场管理后台-会员-配送会员

二、新增单个会员

  1. 在左上角点“新增会员”按钮

  2. 添加会员基本信息

    可透支额度（元）：指套餐余额消费完后还可以透支消费的金额

  3. 设置套餐信息

    单次配送要求：随套餐的要求而变化

    配送日：一个会员至少对应一个配送日和一条配送线路，可设置多个配送日。

    套餐附加商品：指每次点菜前，如果该附加商品有库存，系统会自动把该商品分配到会员的购物车里。

三、批量导入会员

在“配送会员”页左上角点“导入”，按导入指引和模板提醒导入即可

新增会员配送日

发表于 2016-01-21   |   分类于 会员管理

“会员配送日”是指农场为会员设置的固定配送日，例如农场每周二、周五为会员配送，则会员配送日为周二和周五。

一、操作路径

农场管理后台-会员-会员配送日

二、操作流程

  1. 新增配送日
  2. 新增配送线路

三、流程介绍

  1. 新增配送日

    农场至少应该有一个配送日，否则无法添加会员和进行配送；

    配送日:可以从星期一至星期日之间选择，不能重复添加；

    预计点菜开始时间：指农场预计在配送日的前几天几点开放点菜（此处的时间仅供会员在app上参考，真正开始点菜需要农场执行“开始点菜”操作）

    预计点菜结束时间：同上

  2. 新增配送线路

    配送线路是指该配送日一共分几条线配送，每条线路分别采用什么配送方式配送，一个配送日至少需要添加一条线路。

    配送方式有以下三种可选：

    自有物流：即农场自已有司机负责配送，首次添加时需“新增司机”

    好农场合作物流：与好农场后台进行了物流接口对接的物流公司

    第三方物流：即申通、圆通等快递公司，一条线路只能对应一个快递公司

【教程】首次会员配送流程

发表于 2016-01-21   |   分类于 新手指引

如果你是第一次使用好农场来进行会员配送，那么你只需要按照下面的流程一步步操作，就可以完成一次会员配送了。

操作流程

  1. 新增会员
  2. 设置会员菜单
  3. 会员点菜
  4. 开始配送

流程介绍

  1. 新增会员
    新增会员之前，需要先 新增会员配送日 和 会员套餐，完成以上操作后，可以进行 新增会员 操作

  2. 设置会员菜单

    设置会员菜单之前，需要先在商品库中 新增商品（蔬菜或附加商品），新增商品后，可以 设置会员可选菜单和附加商品

  3. 会员点菜

    不管农场是否允许会员点菜，农场需要先找到对应的配送日，并“开始配送计划”。配送计划生成后，农场就可以 开放点菜 了，农场开放点菜后，会员的app就会收到点菜提醒，并开始点菜。到截止时间后，农场可以 结束点菜。

  4. 开始配送

    结束点菜后，就可以打印送货单或销售单并 开始配送 了。配送完成 后，系统会将该配送日的订单批量处理为“配送完成”，整个配送过程就结束了，农场可以到“全部会员订单”中查询历史订单。

好农场零售功能使用须知

发表于 2016-01-21   |   分类于 好农场简介
Q：好农场零售功能的定位是什么样的？目标消费群体是哪些？

A：好农场零售功能的定位是一个以本地农场为主的生态农产品电商平台，软件根据消费者所在位置自动向消费者推荐周边的农场和农产品。目标消费群体是城市里对食品安全有需求的人群（如孕妇、有老人或小孩的家庭、年轻白领等）

Q：好农场零售平台农场入驻的标准或要求是什么？

A：主要有以下几点要求：

  1. 愿意公开并承诺农场的生产方式，例如种养植标准、加工标准；
  2. 接受好农场工作人员或消费者随时到农场参观考察，对生产进行监督；
  3. 可以独立发货、并能够保证向消费者提供有品质保障的农产品；
Q：好农场电商平台和一般的电商有什么区别？流量从哪里来？

A：好农场电商平台是一个只做生态农产品的平台，旨在帮助用心做农业的农人获得合理的回报。平台上的每一个农产品都会有对应的生产者及生产承诺，好农场会根据农场做出的承诺随机抽检，消费者也可以对违反承诺的商品进行举报。好农场平台的流量来自于线下的口碑传播及我们在农夫市集及其它场所做的宣传和推广活动。

好农场支付方式及结算规则

发表于 2016-01-21   |   分类于 好农场简介

一、支付方式

好农场目前支持以下两种在线支付方式：

  1. 微信支付

  2. 支付宝支付

    农场会员充值或消费者购买商品时，可以选择以上方式向农场支付，支付金额统一进入好农场结算账户，农场通过管理后台发起提现申请与好农场结算款项。好农场将给农场补贴支付时产生的手续费用0.6%以及提现时的手续费用。

    说明：目前微信支付和支付宝支付的商户结算费率均为 0.6%
    点击查看： 微信支付结算费率 ， 支付宝支付结算费率

（注：农场会员宅配系统中的会员订单均只能使用会员的储值余额支付，余额不足时需进行充值，与上述的支付方式为不同类型）

二、会员充值或买家下单后，资金如何结算？如何提现？

  1. 会员充值成功后，在“农场”——“收入/提现”里会产生一条充值记录，充值的金额将实时进入农场的账户余额中，农场可以随时申请提现。

  2. 买家购买零售商品并支付后，以下两种情况满足任意一种结算金额可以入账提现：

    • 农场在订单发货T+7天后交易金额将结算入账可提现（从发货当天开始计算，7天结算精确到天）

      例：交易订单在7月17号发货，则7月24号交易金额结算入账。

    • 农场在订单发货后，买家在好农场app上进行了“确认收货”操作，交易金额将结算入账可提现。

      例：交易订单在7月17号发货，7月20号买家确认收货，则交易金额结算入帐。

  3. 提现前农场需添加结算账户，具体查看：如何进行提现操作？ ；提现周期为T+2个工作日，具体到账时间以银行的处理时间为准。
