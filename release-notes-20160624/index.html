2016-06-24 更新：微信端支持扫码支付

发表于 2016-06-24   |   分类于 更新说明

更新要点

  • 【微信端】支持微信扫码支付，解决在订阅号无法支付的问题；

累积更新

  • 【管理系统】管理系统左上角logo换成农场logo以增加识别度，而链接改成指向管理系统首页
  • 【微信端】去掉了头部的导航条；

微信扫码支付

支持微信扫码支付渠道，解决订阅号中无法完成微信支付的问题.

iOS设备会报告此错误（点击确定），Android设备可能无此报错，同时不能弹出微信支付的界面。

显示二维码，手指长按二维码。

点击“识别图中二维码”。

进行正常微信支付流程，完成之后，点击二维码下方的我已完成支付按钮返回。
